# Markup Enhancements

!! JSPWiki enhances wiki beyond Wiki Markup pages through 
* WikiPlugins
* WikiVariables
* WikiTags
* JSPWikiStyles
* InterWiki Links
* Filters
* WikiFroms
* Page Metadata
* Inline Images and Inline Attachments
* Search Enhancements
* RSS and Atom Feeds
which provide dynamic content, customization, and styling.
