# Overview

Developer documentation has been moved to GitHub for better version control and accessibility.

## Developer Documentation Location

All developer documentation is now maintained in the GitHub repository:

**[https://github.com/jwilleke/amdWiki/tree/master/docs](https://github.com/jwilleke/amdWiki/tree/master/docs)**

### Key Documentation

- **[CONTRIBUTING.md](https://github.com/jwilleke/amdWiki/blob/master/CONTRIBUTING.md)** - How to contribute to amdWiki
- **[docs/architecture/](https://github.com/jwilleke/amdWiki/tree/master/docs/architecture)** - Architecture documentation
- **[docs/testing/](https://github.com/jwilleke/amdWiki/tree/master/docs/testing)** - Testing guides
- **[docs/developer/](https://github.com/jwilleke/amdWiki/tree/master/docs/developer)** - Developer guides and API documentation

## Why GitHub?

Developer documentation belongs in the code repository because:
- Always in sync with the codebase
- Version controlled with code changes
- Accessible without running the wiki
- Standard open-source practice
