# Calendar Help

This page explains how to use the community calendar system.

## Viewing the Calendar

Navigate to the [Calendar](Calendar) page. Use the toolbar buttons to switch between views:

| View | Description |
|------|-------------|
| Month | Full month grid — best overview |
| Week | Week with time slots |
| Day | Single day detail |
| List | Scrollable event list |

Click **Today** to jump back to the current date. Use the **‹** and **›** arrows to move forward and backward.

Click any event to see its full details.

## Types of Calendars

| Calendar | Who can view | How events are added |
|----------|-------------|----------------------|
| Community Events | Everyone | Added by staff/administrators |
| Clubhouse | Everyone | Submit a reservation request |
| Tennis Court | Logged-in members | Submit a reservation request |

## Subscribing to the Calendar

You can subscribe to any calendar from Apple Calendar, Google Calendar, Outlook, or any app that supports `.ics` feeds.

Use the subscription URL:

```
http://<your-site>/api/calendar/<calendarId>/feed.ics
```

For example:

- Community Events: `/api/calendar/events/feed.ics`
- Clubhouse: `/api/calendar/clubhouse/feed.ics`

In **Apple Calendar**: File → New Calendar Subscription → paste the URL.

In **Google Calendar**: Other calendars → From URL → paste the URL.

Note: Private reservation details are not included in the public feed.

## Making a Reservation

Members can request time on reservation calendars (Clubhouse, Tennis Court, etc.).

1. Go to the [Make a Reservation](MakeAReservation) page.
2. Fill in the date, time, and purpose.
3. Submit the form.

You will receive a confirmation email. The facility manager will also be notified.

If the time slot is already reserved you will see a conflict message — choose a different time.

## Frequently Asked Questions

**Why can't I see the details of some events?**
Reservation events are private. Only the person who made the reservation and the facility manager can see the full details.

**Can I cancel a reservation I made?**
Yes. Contact the administrator or use the [Make a Reservation](MakeAReservation) page and submit a cancellation note.

**Who do I contact about a calendar error?**
Contact your site administrator.
