# Journal Help

The journal lets you keep private dated entries — like a diary — directly within your pages.

## Getting Started

Click **New Entry** (or visit [/api/journal/new](/api/journal/new)) to create today's entry.  Entries are private by default and only visible to you and administrators.

## Writing an Entry

Journal entries are ordinary pages with a date and optional mood/tags in the frontmatter.  Use the standard editor — markdown, attachments, links, and plugins all work normally.

### Frontmatter fields

| Field | Purpose |
|-------|---------|
| `journal-date` | ISO date (YYYY-MM-DD) — set automatically on creation |
| `mood` | Optional free-text mood label |
| `journal-tags` | Optional YAML list of tags |

## Using the `[{Journal}]` Plugin

Embed a live journal widget on any page:

```
[{Journal}]
[{Journal view='timeline' limit='10'}]
[{Journal view='streak'}]
[{Journal view='on-this-day'}]
```

| Parameter | Values | Default |
|-----------|--------|---------|
| `view` | `timeline`, `streak`, `on-this-day` | `timeline` |
| `limit` | number | `10` |

## Privacy

All entries are stored as private pages (visible only to you and admins).  The author-lock setting prevents others from editing your entries even if they somehow obtain the URL.

## API

| Endpoint | Description |
|----------|-------------|
| `GET /api/journal/new` | Create today's entry and open the editor |
| `GET /api/journal/entries` | JSON list of your entries |
| `GET /api/journal/on-this-day` | Entries from the same calendar date in prior years |
| `GET /api/journal/streak` | Your current writing streak |
