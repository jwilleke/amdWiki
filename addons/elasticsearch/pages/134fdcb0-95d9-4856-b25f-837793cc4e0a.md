# NAS File Search

The NAS File Search lets you find and insert files stored on the network drive directly from the wiki editor — photos, documents, videos, and more.

## Using the Asset Browser

When editing a page, click the **Insert Asset** button (paperclip icon) in the toolbar. The asset browser opens and shows files from all sources including the NAS.

Use the **Source** filter to select **sist2 NAS Index** to show only NAS files.

## Searching for Files

Type in the search box to find files by:

- **Filename** — partial matches work (`vacation` finds `summer-vacation-2023.jpg`)
- **Path** — folder names on the NAS (`family/photos`)
- **Content** — text extracted from documents and images (OCR)
- **Tags** — labels assigned to files

### Filtering Results

| Filter | What it does |
|--------|-------------|
| Type | Limit to Images, Documents, or Other |
| Year | Filter by file modification year |
| Sort | Date modified (newest first) or filename A–Z |

## Inserting a File

Click any result to preview it. Click **Insert** to add it to your page. Image files are inserted as:

```
[{Image src='...'}]
```

Other files are inserted as a link to the full file.

## Thumbnails

Images and some document types show a thumbnail preview. If a thumbnail is not yet available the file was recently added and sist2 is still processing it — try again shortly.

## What Files Are Available

The NAS index includes files from the network-attached storage scanned by sist2. Your administrator controls which folders are visible based on your account role.

## Frequently Asked Questions

**Can I upload files to the NAS from here?**
No. The NAS asset source is read-only. Use your normal file manager or network share to add files to the NAS — they will appear in the wiki after the next scheduled scan.

**How current is the index?**
The NAS is scanned automatically each night. Files added during the day will appear the following morning. Ask your administrator to trigger a manual scan if you need a file to appear sooner.

**I found a file but can't see its thumbnail.**
Thumbnails are generated progressively. If the file was recently indexed the thumbnail may not exist yet. The full file is always available via the link.

**Why can't I see a folder I know exists?**
Access to NAS folders is controlled by your account role. Contact your administrator if you believe you should have access to additional folders.
