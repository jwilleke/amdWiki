# LocationPlugin

The LocationPlugin displays locations with [map] links and optional embedded map previews.

## Description

This plugin creates clickable map links from location names or coordinates. It supports multiple map providers and can optionally embed an interactive map preview directly in the page.

## Syntax

Examples:
[[{Location name='Paris, France'}] renders as:
[{Location name='Paris, France'}]
[[{Location coords='48.8566,2.3522'}] renders as:
[{Location coords='48.8566,2.3522'}]

[[{Location name='Eiffel Tower' embed=true}]  DOES NOT WORK. embed=true requires [Coordinates].
[{Location name='Eiffel Tower' embed=true}]

[[{Location coords='48.8566,2.3522' name='Eiffel Tower' embed=true zoom=15}]
[{Location coords='48.8566,2.3522' name='Eiffel Tower' embed=true zoom=15}]

## Parameters

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| name | string | - | Location name (searched by map service) |
| coords | string | - | Latitude,longitude (e.g., "48.8566,2.3522") |
| embed | boolean | false | Show embedded map preview |
| zoom | number | 13 | Map zoom level (1-18) |
| width | string | "100%" | Embedded map width |
| height | string | "300px" | Embedded map height |
| provider | string | "osm" | Map provider: osm, google, apple, geo |
| label | string | name/coords | Custom display text |

Either `name` or `coords` (or both) must be provided.

## Providers

| Provider | Description |
| --- | --- |
| osm | OpenStreetMap (default, supports embed) |
| google | Google Maps |
| apple | Apple Maps |
| geo | RFC 5870 geo: URI (opens device maps app) |

## Examples

### Location by Name

```wiki
Visit [{Location name='Paris, France'}] for the conference.
```

Creates a link that opens OpenStreetMap searching for "Paris, France".

### Precise Coordinates

```wiki
The summit is at [{Location coords='27.9881,86.9250' label='Mt. Everest'}].
```

Links directly to exact coordinates with custom display text.

### Embedded Map

```wiki
[{Location coords='48.8566,2.3522' name='Eiffel Tower' embed=true zoom=15}]
```

Shows an interactive embedded map preview with the location marker.

### Google Maps Link

```wiki
[{Location coords='40.7128,-74.0060' provider='google' label='NYC'}]
```

Opens Google Maps instead of OpenStreetMap.

### Mobile Deep Link

```wiki
[{Location coords='51.5074,-0.1278' provider='geo' label='London'}]
```

Uses `geo:` URI which opens the device's native maps application.

### Custom Embed Size

[[{Location coords='35.6762,139.6503' embed=true width='600px' height='400px'}] renders as:

[{Location coords='35.6762,139.6503' embed=true width='600px' height='400px'}]

```wiki
[{Location coords='35.6762,139.6503' embed=true width='600px' height='400px'}]
```

## Output

- Without embed: Inline link with map marker icon
- With embed: Container with header link and embedded map
