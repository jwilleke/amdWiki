# Platform Variables

[{$pagename}] is the term used in [{$applicationname}] for dynamic values that can be inserted into wiki pages. These are also known as [WikiVariables|Platform Variables] (the JSPWiki term).

## Variable Types

[{$applicationname}] supports three categories of variables:

### [Configuration Values]

[Configuration Values] come from the system configuration and are accessed using the [ConfigAccessorPlugin]. These are static values set by administrators.

**Syntax:** [[{ConfigAccessor property='amdwiki.property.name'}] will show:

**Examples:**

- Application name
- Base URL
- Feature flags
- System limits

See [Configuration System] for details on available configuration properties.

### [Contextual Variables]

[Contextual Variables] are dynamic values that depend on the **current user session or page** being viewed.

**Syntax:** `[{$variablename}]`

**Examples:**

| Variable | Description | Example Output |
| --- | --- | --- |
| `[{$pagename}]` | Current page title | [{$pagename}] |
| `[{$username}]` | Current user | [{$username}] |
| `[{$requestcontext}]` | Current action | [{$requestcontext}] |

### [System Variables]

[System Variables] are dynamic values about the wiki system itself, independent of user or page context.

**Syntax:** `[{$variablename}]`

**Examples:**

| Variable | Description | Example Output |
| --- | --- | --- |
| `[{$applicationname}]` | Wiki name | [{$applicationname}] |
| `[{$totalpages}]` | Total page count | [{$totalpages}] |
| `[{$uptime}]` | Server uptime | [{$uptime}] |
| `[{$timestamp}]` | Current timestamp | [{$timestamp}] |

## Using Variables

Variables are inserted into wiki pages using the JSPWiki syntax:

```text
The current page is [{$pagename}] and you are logged in as [{$username}].

This wiki ([{$applicationname}]) has [{$totalpages}] pages.
```

## Variable Plugins

Several plugins provide variable functionality:

| Plugin | Purpose |
| --- | --- |
| [VariablesPlugin] | Display all available variables |
| [ConfigAccessorPlugin] | Access configuration values |
| [Current Time Plugin] | Display formatted date/time |
| [UptimePlugin] | Display server uptime |
| [TotalPagesPlugin] | Display page count |

## Viewing Available Variables

Use the [VariablesPlugin] to see all available variables:

```text
[{VariablesPlugin}]
```
