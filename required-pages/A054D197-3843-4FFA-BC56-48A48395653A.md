# Plugins

Plugins extend amdWiki functionality by embedding dynamic content in wiki pages.

## Overview

Plugins are invoked using the JSPWiki-compatible syntax:

```wiki
[{PluginName param='value'}]
```

All plugins support zero or more parameters to customize their output.

## Available Plugins

### Content Display

| Plugin | Description |
| -------- | ------------- |
| [CurrentTimePlugin] | Display current date/time |
| [ImagePlugin] | Embed images with captions |
| [VariablesPlugin] | Display wiki variables |

### Navigation & Indexing

| Plugin | Description |
| -------- | ------------- |
| [IndexPlugin] | Alphabetical page index |
| [ReferringPagesPlugin] | Pages linking to current page |
| [SearchPlugin] | Embedded search results |
| [RecentChangesPlugin] | Recently modified pages |

### System Information

| Plugin | Description |
| -------- | ------------- |
| [SessionsPlugin] | Active session count |
| [TotalPagesPlugin] | Total wiki page count |
| [UptimePlugin] | Server uptime |

### Utility

| Plugin | Description |
| -------- | ------------- |
| [CounterPlugin] | Incrementing counters |
| [ConfigAccessorPlugin] | Access configuration values |

## Quick Reference

### CurrentTimePlugin

```wiki
[{CurrentTimePlugin}]
[{CurrentTimePlugin format='yyyy-MM-dd HH:mm'}]
```

### ImagePlugin

```wiki
[{Image src='photo.jpg' caption='My Photo'}]
[{Image src='diagram.png' align='center' width='400'}]
```

### IndexPlugin

```wiki
[{IndexPlugin}]
[{IndexPlugin include='^Doc.*' exclude='.*Test.*'}]
```

### ReferringPagesPlugin

```wiki
[{ReferringPagesPlugin}]
[{ReferringPagesPlugin max=5 show='count'}]
```

### SearchPlugin

```wiki
[{Search query='keyword' max=10}]
[{Search system-category='documentation' format='titles'}]
```

### SessionsPlugin

```wiki
[{SessionsPlugin}]
[{SessionsPlugin property='distinctUsers'}]
```

### TotalPagesPlugin

```wiki
[{TotalPagesPlugin}]
```

### UptimePlugin

```wiki
[{UptimePlugin}]
```

## Creating Dashboard Pages

Combine plugins to create dynamic dashboards:

```wiki
!! System Status

| Metric | Value |
| -------- | ------- |
| Total Pages | [{TotalPagesPlugin}] |
| Active Sessions | [{SessionsPlugin}] |
| Server Uptime | [{UptimePlugin}] |
| Current Time | [{CurrentTimePlugin}] |
```
