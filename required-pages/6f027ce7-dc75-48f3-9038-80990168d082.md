# User Keywords

**User Keywords** are tags you can assign to a page to help users find it through search and filtering. They are optional.

## How to Use Them

Click the **User Keywords** dropdown on the edit form and check any keywords that apply to the page. The dropdown shows all available keywords defined for this [{$applicationname}] instance.

## Limits

There is a maximum number of keywords allowed per page (shown in the label). Trying to save with more than the maximum will show a validation error.

## How They Affect Search

User keywords are indexed alongside the page title and content. Pages with matching keywords appear in search results when a user searches for that keyword. They can also be used to filter pages in listings and indexes.

## Keywords vs. System Category

User keywords are **user-defined tags** — any editor can add or change them. [System Category] is a **system-controlled classification** that only admins can change.
