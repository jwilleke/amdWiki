# Page Statistics

**Page Statistics** is available from the **Info** dropdown in the toolbar. It opens a modal showing metadata and access statistics for the current page.

Statistics are fetched from `/api/page-metadata/PageName` and displayed in a modal overlay — no page navigation required.
