# Overview

[{$pagename}] is part of the [Keywords and Categories] that determines where pages are stored.

## Simple tier system

### system-category: system

* "storageLocation": "required" (/required-pages)
* Needed for wiki operation
* Listed on [System Pages]

### system-category: developer

* docs/ (GitHub only, NOT in wiki)
* Technical/developer documentation
* Links are in [Documentation for Developers] page

### system-category: documentation

* "storageLocation": "required" (/required-pages)
* User-facing help/guides
* Listed on User [Wiki Documentation] Page

### system-category: general

* "storageLocation": "regular" (/pages)
* Not kept in GitHub

### storageLocation

* "storageLocation": "regular" - Puts the page into "pages"
* "storageLocation": "required"  - Puts the page into "required-pages"
* "storageLocation": "github" - Implies it is not part of Wiki pages
* "default": true - Implies this is the "Default" value for [{$pagename}]
