# SearchPlugin

The **SearchPlugin** embeds search results directly in wiki pages.

## Description

This plugin allows you to display search results within page content. You can search by text, filter by category or keywords, and choose various output formats. Great for creating dynamic index pages.

## Syntax

```wiki
[{Search query='keyword'}]
```

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `query` | string | * | Search text (* for all pages) |
| `system-category` | string | - | Filter by category |
| `user-keywords` | string | - | Filter by keywords (pipe-separated) |
| `max` | number | 50 | Maximum results |
| `format` | string | table | Output: table, count, titles, list |

## Examples

### Basic Text Search

```wiki
[{Search query='plugin' max=10}]
```

Shows pages containing "plugin" in a table with scores.

### All Pages in Category

```wiki
[{Search system-category='documentation'}]
```

Lists all documentation pages.

### Count Format

```wiki
There are [{Search system-category='system' format='count'}] system pages.
```

**Output:** `There are 42 system pages.`

### Titles Format

```wiki
[{Search system-category='documentation' format='titles'}]
```

Shows a bullet list with page links.

### Multiple Keywords (OR)

```wiki
[{Search user-keywords='economics|geology'}]
```

Pages tagged with economics OR geology.

## Output Formats

| Format | Description |
|--------|-------------|
| table | Full table with page names and scores (default) |
| count | Just the number of results |
| titles | Bullet list with links |
| list | Simple list without links |

## Use Cases

- **Category indexes**: Automatically list all pages in a category
- **Topic pages**: Show all pages about a subject
- **Dashboards**: Display counts and quick links
