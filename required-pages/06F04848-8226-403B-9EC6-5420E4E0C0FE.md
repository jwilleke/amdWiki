# TotalPagesPlugin

The **TotalPagesPlugin** displays the total number of wiki pages.

## Description

This simple plugin returns a count of all pages in the wiki. It's useful for statistics, dashboards, and welcome pages.

## Syntax

```wiki
[{TotalPagesPlugin}]
```

## Parameters

This plugin has no parameters.

## Examples

### Simple Count

```wiki
This wiki contains [{TotalPagesPlugin}] pages.
```

**Output:** `This wiki contains 247 pages.`

### Welcome Page

```wiki
Welcome to our wiki! Browse our [{TotalPagesPlugin}] pages of documentation.
```

### Statistics Dashboard

```wiki
| Metric | Value |
| -------- | ------- |
| Total Pages | [{TotalPagesPlugin}] |
| Active Users | [{SessionsPlugin}] |
| Server Uptime | [{UptimePlugin}] |
```

## Output

Returns a plain number (e.g., "247") suitable for embedding in sentences.
