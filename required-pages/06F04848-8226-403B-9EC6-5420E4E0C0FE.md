# TotalPagesPlugin

The **TotalPagesPlugin** displays the total count of pages on the platform. It is useful for statistics dashboards, welcome pages, and any page where you want to show how many pages exist. See [Plugins] for a complete list of available plugins.

## Syntax

```
[{TotalPagesPlugin}]
```

## Parameters

This plugin has no parameters.

## Examples

### Simple Count

```
This platform has [{TotalPagesPlugin}] pages.
```

**Output:** `This platform has 247 pages.`

### Statistics Dashboard

```
| Metric | Value |
| -------- | ------- |
| Total Pages | [{TotalPagesPlugin}] |
| Active Sessions | [{SessionsPlugin}] |
| Server Uptime | [{UptimePlugin}] |
```

## Output

Returns a plain number (e.g., `247`) suitable for embedding in sentences or tables.
