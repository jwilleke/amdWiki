# VariablesPlugin

The **VariablesPlugin** displays system and contextual variables available in the wiki.

## Description

This plugin shows all available variables that can be used throughout the wiki, including system information, user context, and available plugins.

## Syntax

```wiki
[{VariablesPlugin}]
```

## Parameters

| Parameter | Type | Default | Description |
| ----------- | ------ | --------- | ------------- |
| `type` | string | `all` | What to display: `all`, `system`, `contextual`, or `plugins` |

## Examples

### Display All Variables (Default)

```wiki
[{VariablesPlugin}]
```

**Output:** Tabbed interface with System Variables, Contextual Variables, and Available Plugins

### System Variables Only

```wiki
[{VariablesPlugin type='system'}]
```

Shows variables like:

- `[{$applicationname}]` - Application name
- `[{$version}]` - amdWiki version
- `[{$baseurl}]` - Base URL
- `[{$uptime}]` - Server uptime
- `[{$totalpages}]` - Total page count

### Contextual Variables Only

```wiki
[{VariablesPlugin type='contextual'}]
```

Shows variables like:

- `[{$username}]` - Current user
- `[{$pagename}]` - Current page
- `[{$browser}]` - Browser info
- `[{$clientip}]` - Client IP address

### Available Plugins Only

```wiki
[{VariablesPlugin type='plugins'}]`
```

Shows all registered plugins with descriptions.

## Variable Types

### System Variables

System-level information (no user context required):

- Application settings (`applicationname`, `version`, `baseurl`)
- Statistics (`totalpages`, `uptime`)
- Date/time (`date`, `time`, `year`, `month`, `day`)

### Contextual Variables

User and request-specific information:

- User info (`username`, `displayname`, `loginstatus`, `userroles`)
- Page info (`pagename`)
- Request info (`browser`, `clientip`, `referer`, `sessionid`)

## Common Use Cases

### Variables Reference Page

```wiki
# Available Variables

See all variables you can use in pages:

[{VariablesPlugin}]
```

### System Info Dashboard

```wiki
## System Information

[{VariablesPlugin type='system'}]
```

### Debug Page

```wiki
# Debug Information

## Your Session
[{VariablesPlugin type='contextual'}]

## Available Plugins
[{VariablesPlugin type='plugins'}]
```

## Using Variables in Pages

Once you know the available variables, use them in any page:

```wiki
Welcome [{$username}]!

This wiki has [{$totalpages}] pages.
```
