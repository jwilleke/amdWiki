# VariablesPlugin

The **VariablesPlugin** displays system and contextual variables available in the wiki.

## Description

This plugin shows all available variables that can be used throughout the wiki, including system information, user context, and available plugins.

## Syntax

```wiki
[{VariablesPlugin}]
```

## Parameters

| Parameter | Type | Default | Description |
| ----------- | ------ | --------- | ------------- |
| `type` | string | `all` | What to display: `all`, `system`, `contextual`, or `plugins` |

## Examples

### Display All Variables (Default)

[[{VariablesPlugin}] will show the following:
[{VariablesPlugin}]

**Output:** Tabbed interface with System Variables, Contextual Variables, and Available Plugins

### System Variables Only

[System Variables] Only
[[{VariablesPlugin type='system'}]  will show the following:
[{VariablesPlugin type='system'}]

### Contextual Variables Only

[Contextual Variables] Only
[[{VariablesPlugin type='contextual'}] will show the following:
[{VariablesPlugin type='contextual'}]

### Available Plugins Only

Available [Plugins] Only
[[{VariablesPlugin type='plugins'}] will show Shows all registered [plugins] with descriptions:
[{VariablesPlugin type='plugins'}]

## Common Use Cases

Once you know the available variables, use them in any page:

```wiki
Welcome [{$username}]!

This wiki has [{$totalpages}] pages.
```
