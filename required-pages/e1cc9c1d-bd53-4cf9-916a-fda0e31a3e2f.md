# Page Information

**Page Information** is available from the **Info** dropdown in the toolbar. It opens a modal showing the current page's metadata.

## What It Shows

|| Field || Description ||
| Title | The page's display name |
| Page Name | The internal name used in URLs |
| Slug | The URL-friendly identifier |
| UUID | The page's unique system identifier |
| Version | Whether the page is versioned |
| URL | The direct link to the page |
| Created | Date and time the page was first created |
| Last Modified | Date and time of the most recent save |
| Last Accessed | Date and time of the most recent view |
| Author | The user who originally created the page |
| Category | The [System Category] of the page |

## Also in the Info Dropdown

|| Item || Description ||
| [Page History] | View the revision history of the page |
| Page Source | View the raw markup source |
| [Page Statistics] | View access and metadata statistics |
| [Recent Changes] | View recently changed pages across the platform |
