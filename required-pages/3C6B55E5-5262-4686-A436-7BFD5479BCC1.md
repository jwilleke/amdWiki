# SessionsPlugin

The **SessionsPlugin** displays the number of active user sessions.

## Description

This plugin shows how many users are currently connected to the wiki. It can display either total sessions or distinct logged-in users.

## Syntax

```wiki
[{SessionsPlugin}]
```

## Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `property` | string | users | What to count: "users" or "distinctUsers" |

## Examples

### Total Sessions

```wiki
Currently [{SessionsPlugin}] active sessions.
```

**Output:** `Currently 5 active sessions.`

### Distinct Users

```wiki
[{SessionsPlugin property='distinctUsers'}] users online.
```

**Output:** `3 users online.`

### Status Dashboard

```wiki
| Metric | Value |
|--------|-------|
| Active Sessions | [{SessionsPlugin}] |
| Unique Users | [{SessionsPlugin property='distinctUsers'}] |
| Server Uptime | [{UptimePlugin}] |
```

## Output

Returns a plain number suitable for embedding in text.
