# Using SessionsPlugin

The **SessionsPlugin** displays the number of active user sessions. It can show either total sessions or distinct logged-in users. See [Plugins] for a complete list of available plugins.

## Syntax

[[{SessionsPlugin}] renders as:

[{SessionsPlugin}]

## Parameters

%%table-striped
|| Parameter || Type || Default || Description ||
| `property` | string | `users` | What to count: `users` (total sessions) or `distinctUsers` (unique logged-in users) |
/%

## Examples

### Total sessions

[[{SessionsPlugin}] active sessions.] renders as:

[{SessionsPlugin}] active sessions.

### Distinct logged-in users

[[{SessionsPlugin property='distinctUsers'}] unique users online.] renders as:

[{SessionsPlugin property='distinctUsers'}] unique users online.
