# SessionsPlugin

The **SessionsPlugin** displays the number of active user sessions. It can show either total sessions or distinct logged-in users. See [Plugins] for a complete list of available plugins.

## Syntax

```
[{SessionsPlugin}]
```

## Parameters

| Parameter | Type | Default | Description |
| ---- | ---- | ---- | ---- |
| `property` | string | `users` | What to count: `users` (total sessions) or `distinctUsers` (unique logged-in users) |

## Examples

### Total Sessions

```
Currently [{SessionsPlugin}] active sessions.
```

**Output:** `Currently 5 active sessions.`

### Distinct Users

```
[{SessionsPlugin property='distinctUsers'}] users online.
```

**Output:** `3 users online.`

### Status Dashboard

```
| Metric | Value |
| -------- | ------- |
| Active Sessions | [{SessionsPlugin}] |
| Unique Users | [{SessionsPlugin property='distinctUsers'}] |
| Server Uptime | [{UptimePlugin}] |
| Total Pages | [{TotalPagesPlugin}] |
```

## Output

Returns a plain number suitable for embedding in text or tables.
