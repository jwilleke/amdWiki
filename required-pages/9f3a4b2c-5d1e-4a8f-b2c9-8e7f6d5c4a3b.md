# RecentChangesPlugin

The **RecentChangesPlugin** displays recent page changes in chronological order, helping you track wiki activity.

## Description

This plugin shows which pages have been modified recently. It supports two display formats: a compact list for quick scanning or a detailed table with full information.

## Syntax

```wiki
[{RecentChangesPlugin}]
```

## Parameters

| Parameter | Type | Default | Description |
| ---- | ---- | ---- | ---- | ---- |
| `since` | number | `7` | Number of days to look back for changes |
| `format` | string | `compact` | Display format: `compact` or `full` |

## Examples

### Last 7 Days (Default)

```wiki
[{RecentChangesPlugin}]
```

**Output:** Compact list of pages changed in the last week

### Last 24 Hours

```wiki
[{RecentChangesPlugin since='1'}]
```

### Detailed View

```wiki
[{RecentChangesPlugin since='7' format='full'}]
```

**Output:** Table with page, date, author, and version

### Recent Activity Dashboard

```wiki
## What's New Today
[{RecentChangesPlugin since='1' format='full'}]

## This Week
[{RecentChangesPlugin since='7'}]
```

## Output Formats

### Compact Format (Default)

- Page title (linked)
- Relative timestamp ("2 hours ago", "3 days ago")
- Total change count

Example:

```
Recent Changes (Last 7 days)
• Home (2 hours ago)
• Getting Started (5 hours ago)
• Configuration (1 day ago)

3 changes
```

### Full Format

- Detailed table with columns:
  - Page (linked)
  - Last Modified (date/time)
  - Author
  - Version

## Common Use Cases

### Recent Changes Page

```wiki
# Recent Changes

See what's been updated recently:

[{RecentChangesPlugin since='30' format='full'}]
```

### Sidebar Widget

```wiki

```

### Team Dashboard

```wiki

[{RecentChangesPlugin since='1' format='full'}]
```
