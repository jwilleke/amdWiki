# Configuration Values

**Configuration Values** are [WikiVariables] that are read from the [Configuration System] of [{$applicationname}]. They are provided by [Plugins] and allow significant features to be customized.

## Usage

Access configuration values using the [ConfigAccessorPlugin]:

```text
[{ConfigAccessor property='ngdpbase.applicationname'}]
```

See [Platform Variables] for an overview of all variable types.
