# Overview

[{$pagename}] are [WikiVariables] that are read from the [Configuration System] of [{$applicationname}].

[{$pagename}] are provided from [Plugins] and allow significant features to be customized.

## Usage

Access configuration values using the [ConfigAccessorPlugin]:

```text
[{ConfigAccessor property='amdwiki.applicationname'}]
```
