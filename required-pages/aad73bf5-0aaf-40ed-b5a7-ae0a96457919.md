# Overview

[{$pagename}] operations are performed by [Administrator]

## [{$pagename}] Configuration

If "amdwiki.backup.autoBackup" key is: true

- then you can expect backups to happen automatically and be stored in  [{ConfigAccessorPlugin key="amdwiki.backup.directory", valueonly=true}].

If "amdwiki.backup.autoBackup" key is: false

- then no auto backups are performed.

[Administrators] can manually initiate a [{$pagename}] at [Administrator Dashboard Backup Data|/admin|target='_blank']

[{$pagename}] [Configuration System] has the following values:
[{ConfigAccessorPlugin key="amdwiki.backup*"}]
