# ReferringPagesPlugin

The **ReferringPagesPlugin** displays a list of pages that link to the current page (backlinks).

## Description

This plugin helps you discover which pages reference the current page, creating a "backlinks" or "what links here" feature. It is useful for understanding page relationships and finding related content. See [Plugins] for a complete list of available plugins.

## Syntax

```
[{ReferringPagesPlugin}]
```

## Parameters

All parameters are optional:

| Parameter | Type | Default | Description |
| ----------- | ------ | --------- | ------------- |
| `page` | string | current page | Show references to this page instead |
| `max` | number | 10 | Maximum pages to display |
| `show` | string | list | Output mode: "list" or "count" |
| `before` | string | - | Text before each link |
| `after` | string | - | Text after each link |

## Examples

### Basic List

```
[{ReferringPagesPlugin}]
```

Shows a bulleted list of up to 10 pages linking to this page.

### Count Only

```
This page is referenced by [{ReferringPagesPlugin show='count'}] pages.
```

**Output:** `This page is referenced by 5 pages.`

### Limited Results

```
[{ReferringPagesPlugin max=3}]
```

Shows at most 3 referring pages.

### Different Page

```
Pages linking to Main:
[{ReferringPagesPlugin page='Main'}]
```

### Bullet List Format

```
[{ReferringPagesPlugin before='* ' after='\n'}]
```

Creates a formatted bullet list.

## Output

- **List mode**: Displays a `<ul>` list of clickable page links
- **Count mode**: Returns just the number of referring pages
- **No references**: Shows "No pages currently refer to this page"

## Use Cases

- **Navigation**: Help users discover related content
- **Content management**: Find orphaned or well-connected pages
- **See Also sections**: Automatically generate related links
