# AttachPlugin

The **AttachPlugin** renders [Attachment|AttachmentManager] files inline in page content. See [Plugins] for a complete list of available plugins. Image attachments display as clickable thumbnails; all other files display as styled download links.

## Basic Syntax

```
[{ATTACH src='filename.pdf'}]
[{ATTACH src='photo.jpg' caption='My Photo'}]
```

Or using the positional form (legacy JSPWiki syntax):

```
[{ATTACH filename.pdf}]
[{ATTACH photo.jpg|Caption Text}]
```

## Parameters

### Required Parameters

| Parameter | Description | Example |
| ----------- | ------------- | --------- |
| `src` | Filename, URL, or `media://filename` for media library items | `src='report.pdf'` |

### Optional Parameters

| Parameter | Description | Example | Default |
| ----------- | ------------- | --------- | --------- |
| `caption` | Caption text (also used as alt text for images) | `caption='Q4 Report'` | Filename |
| `align` | Alignment | `align='left'`, `align='right'`, `align='center'` | None |
| `display` | Image display mode (images only) | `display='float'`, `display='block'`, `display='inline'`, `display='full'` | `block` |
| `style` | Custom inline CSS | `style='border-radius: 8px;'` | None |
| `class` | Custom CSS class | `class='featured-doc'` | None |
| `target` | Link target | `target='_blank'` | `_blank` for files |
| `width` | Image width (images only) | `width='300'` | Original |
| `height` | Image height (images only) | `height='200'` | Original |

## Image Attachments

When the attachment is an image (`.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`, `.svg`, `.bmp`), the plugin renders a clickable thumbnail that links to the full-size file.

### Display Modes

The `display` parameter controls how the image interacts with surrounding content:

### `display='block'` (Default)

Image displayed as its own block with no text wrapping.

```
[{ATTACH src='diagram.jpg' align='center' display='block' caption='System Architecture'}]
```

### `display='float'`

Image floats to one side; text wraps around it. Use with `align='left'` or `align='right'`.

```
[{ATTACH src='photo.jpg' align='left' display='float' caption='Team Photo'}]

This text wraps around the image on the right side.
```

### `display='inline'`

Image flows inline with surrounding text — useful for icons.

```
Click the [{ATTACH src='icon.png' display='inline' alt='edit'}] button to continue.
```

### `display='full'`

Full-width image spanning the entire container.

```
[{ATTACH src='banner.jpg' display='full' caption='Welcome Banner'}]
```

## File Attachments

For all non-image files, the plugin renders a download link with a type-specific icon:

```
[{ATTACH src='report.pdf' caption='Download the Report'}]
[{ATTACH src='data.xlsx' caption='Spreadsheet Data'}]
[{ATTACH src='archive.zip'}]
```

Supported icon types: PDF, Word, Excel, PowerPoint, archive, audio, video, text, and generic.

## Alignment

| Value | Description |
| ------- | ------------- |
| `left` | Align left (or float left in float mode) |
| `right` | Align right (or float right in float mode) |
| `center` | Center the image |

## Common Use Cases

### Embedded PDF Download Link

```
[{ATTACH src='mwg_guidance.pdf' caption='MWG Guidance Document'}]
```

### Floating Article Image

```
[{ATTACH src='article-photo.jpg' align='left' display='float' caption='Figure 1: Results' width='300'}]

The study found that...
```

### Centered Standalone Image

```
[{ATTACH src='diagram.png' align='center' display='block' caption='Architecture Diagram'}]
```

### Image with Custom Styling

```
[{ATTACH src='photo.jpg' style='border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.2);' caption='Styled Photo'}]
```

### Linked Document with Custom Class

```
[{ATTACH src='policy.pdf' caption='Privacy Policy' class='policy-link' target='_blank'}]
```

## How Files Are Resolved

`src` is resolved in this order:

| Step | Trigger | Behaviour |
| ---- | ------- | --------- |
| 1 | `src` starts with `media://` | Looked up in the [Media] library by filename — no upload needed |
| 2 | `src` starts with `http://`, `https://`, or `/` | Used as-is |
| 3 | Plain filename | Current page's attachments (exact filename match) |
| 4 | Plain filename | Global attachment search across all pages |
| — | No match | `[Attachment not found: filename]` shown instead |

### Embedding a Media Library Photo

If the [Media] feature is enabled, you can reference library photos without uploading them:

```
[{ATTACH src='media://IMG_1234.jpg' align='left' display='float' caption='Family Trip 2024'}]
[{ATTACH src='media://vacation.jpg'}]
```

## Troubleshooting

### "Attachment not found"

1. Verify the filename matches exactly (case-sensitive)
2. Confirm the file has been uploaded to this page or another page
3. Try uploading the file again via the attachments panel

### Image Not Displaying

1. Check that the file extension is a supported image type (`.jpg`, `.png`, `.gif`, `.webp`, `.svg`, `.bmp`)
2. Verify the file was uploaded successfully
3. Try `[{ATTACH src='filename.jpg' display='block'}]` to rule out float/layout issues

### Text Not Wrapping Around Image

1. Ensure `display='float'` is set
2. Ensure `align='left'` or `align='right'` is set
3. Check that there is enough surrounding text to wrap

See [Attachments] for uploading files, [ImagePlugin] for the image-only plugin with the same display options, and [Media] for the media library.
