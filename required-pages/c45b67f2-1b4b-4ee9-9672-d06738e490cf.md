# Author Lock

**Author Lock** is a flag that restricts editing of a page to its original author and administrators. All other authenticated users can view the page but cannot open the edit form.

## What It Does

When Author Lock is enabled on a page:

- Only the **page author** and **administrators** can edit it
- All other roles (Editor, Contributor, Reader, Occupant) are blocked from editing
- The page remains **visible** to any role that has view access

## Who Can Set or Remove It

Only **administrators** can check or uncheck the Author Lock field. The field is not shown to non-admin users.

## Common Uses

- **Profile pages** — each user's profile page is author-locked by default so only that user (and admins) can modify it
- **Policy or reference pages** — pages that should not be casually edited
