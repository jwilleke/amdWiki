# SystemInfo

**SystemInfo** demonstrates JSPWiki-style system variables that work on any page.

| Date: | [[{$date}] | [{$date}] |
| Time: | [[{$time}] | [{$time}] |

These system variables are processed by the RenderingManager and can be used on **any page** on the platform.
