# Table Syntax Examples

Live examples of table markup syntax. For the full reference, see [JSPWiki Table Functionality].

## Striped Table

%%table-striped
|| Name || Age || City ||
| Alice | 25 | New York |
| Bob | 30 | Chicago |
| Charlie | 35 | Los Angeles |
| Diana | 28 | Seattle |
/%

## Styled Table (Column Color)

[{Table evenRowStyle:'background: lightblue;'}]
|| Product || Price ||
| Item A | $10 |
| Item B | $20 |
| Item C | $15 |

## Auto-Numbered Rows

[{Table}]
||# || Task ||
|# | First task |
|# | Second task |
|# | Third task |
