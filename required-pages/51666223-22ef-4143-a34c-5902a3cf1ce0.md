# Editing a Page

To edit a page in [{$applicationname}], click the **Edit** button in the page toolbar. You must be logged in and have [permission] to edit the page.

The edit form contains the following fields — click any field name for detailed help:

| Field | Who Can Edit |
|---|---|
| [Page Title|/view/page-title] | All editors |
| [System Category] | Admins only |
| [Author Lock] | Admins only |
| [User Keywords] | All editors |
| [Page Audience] | All editors |
| [Page Content] | All editors |
| [Page Attachments] | All editors |

## Saving and Cancelling

- Click **Save Page** to save your changes. The page will be updated immediately.
- Click **Cancel** to discard your changes and return to the page view.

## Section Editing

Section Editing allows you to edit a single section of a page by clicking the **Edit** link next to a section heading. This opens the same edit form but only loads that section's content. Click **Edit full page** in the form to switch back to editing the whole page.
